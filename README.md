# BetterNether
BetterNether Mod for Fabric, MC 1.15.1

Building:
* Clone repo to some folder
* Download "fabric mod example" into separate folder with name "BetterNether"
* Replace all files (except folders) to same from repo
* Run command line in folder (mod example): gradlew genSources eclipse
* Copy all except src to repo folder
* Delete mod example folder
* Rename mod folder to "BetterNether"
* Import project to eclipse
* Mark project as maven
* Disable filter for run configurations
